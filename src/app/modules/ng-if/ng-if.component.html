Nullish Coalescing (??) Operator Kod Örneği: const userName = retrievedUserName ?? 'Misafir';

Eğer "retrievedUserName" değişkeni null veya undefined ise, "userName" değişkeninin çıktısı 'Misafir' olacaktır.

Ama "retrievedUserName" değişkeni null veya undefined değilse, "userName" değişkeninin değeri "retrievedUserName" olacaktır.


Logical OR (||) Operator Kod Örneği: const userName = retrievedUserName || 'Misafir';

Eğer "retrievedUserName" değişkeni null, undefined veya boş string ise, "userName" değişkeninin çıktısı 'Misafir' olacaktır.

Aksi takdirde, "retrievedUserName" değeri "userName" değişkenine atanır.


Logical AND (&&) Operator Kod Örneği: const result = value1 && value2;

Eğer sol tarafındaki değer true ise, sağ taraftaki değeri döndürür; aksi halde sol tarafındaki değeri döndürür.
